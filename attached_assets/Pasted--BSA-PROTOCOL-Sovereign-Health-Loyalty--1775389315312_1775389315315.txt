Промпт: Создай экосистему "BSA.PROTOCOL - Sovereign Health & Loyalty"

Концепция: Это не платежная система, а децентрализованный протокол управления медицинскими скидками. Пользователь "майнит" здоровье, а накопленные баллы (в SOL на Devnet) тратит исключительно на медицинские услуги в партнерской сети.

Стиль интерфейса: Obsidian Futuristic. Глубокий черный фон, стекломорфизм (glassmorphism), неоновые зеленые (#00FF41) и бирюзовые акценты. Шрифты моноширинные для тех-эффекта. Мобильно-адаптивный дизайн.

Основные модули для реализации:

1. Личный кабинет (Sovereign Wallet):

Отображение баланса SOL в Solana Devnet как "Мед-баланс BSA".

Плашка: "Целевой баланс: Заморожено для медицины". Пояснение: "SOL заблокированы смарт-контрактом для использования в клиниках-партнерах".

Калькулятор выгоды: "Сэкономлено на лечении за месяц: [X] ₸".

2. Биометрический Центр (AI Oracle):

Виджеты: Пульс, Шаги, Сон (симуляция данных).

Кнопка "Анализировать витальные показатели".

Логика ИИ (Gemini 2.5 Flash): Анализирует данные и выдает текстовый вердикт (Explainability). Если балл > 80, начисляется награда 0.1 SOL как "Медицинский дисконт".

3. Маркетплейс услуг (Circular Economy):

Список клиник. Флагман — "Tibora Центр восточной медицины (Астана)".

Карточки услуг: "Первичная консультация", "Акупунктура (5 сеансов)", "Диагностика Pulsoma".

Визуализация цены: Старая цена (перечеркнута) и новая "Цена BSA" со скидкой 50%.

Кнопка: "Активировать скидку за [X] SOL". При нажатии происходит транзакция с кошелька пользователя на кошелек клиники.

4. Юридическая безопасность (Compliance):

Везде используй термин "Активация скидки" или "Бонусные баллы" вместо "Оплата".

Добавь блок "Правила протокола": "Начисления являются целевыми медицинскими ваучерами в рамках программы лояльности QazTechMedicus".

5. Технический стек:

React + Tailwind v4 + shadcn/ui.

Backend: Node.js (Express) для связи с Gemini API и Solana.

Blockchain: Solana Web3.js (@solana/wallet-adapter-react).

AI: Google Generative AI SDK (Gemini 2.5 Flash).

Миссия проекта: Создать замкнутый цикл, где дисциплина пользователя конвертируется в доступное и качественное лечение в клинике Tibora.
